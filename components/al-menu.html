    # 3. solt

    al-menu 内置 menuItem/submenu 插槽，具体使用方法请参考示例

    • 用户（路径：/user）（点击跳转）
    • 管理（路径：/admin）（点击展开）
      • vip1（路径：/admin/vip1）（点击展开）
        • vip11（路径：/admin/vip1/vip1-1）（点击跳转）
        • vip12（路径：/admin/vip1/vip1-2）（点击跳转）
      • vip2（路径：/admin/vip2）（点击跳转）
      • vip3（路径：/admin/vip3）（点击跳转） 
    <template>
  <al-menu :menuList="menuList" router>
    <template v-slot:menuItem="scope">
      {{scope.row.name}}（路径：{{scope.row.path}}）（点击跳转）
    </template>
    <template v-slot:submenu="scope">
      {{scope.row.name}}（路径：{{scope.row.path}}）（点击展开）
    </template>
  </al-menu>
</template>

<script>
export default {
  data() {
    return {
      menuList: [
        {
          path: "/user",
          name: "用户",
          meta: { icon: "el-icon-mobile" },
        },
        {
          path: "/admin",
          name: "管理",
          meta: { icon: "el-icon-eleme" },
          children: [
            {
              path: "/admin/vip1",
              name: "vip1",
              children: [
                {
                  path: "/admin/vip1/vip1-1",
                  name: "vip11",
                },
                {
                  path: "/admin/vip1/vip1-2",

            },
            {

              name: "vip2",
            },
            {
              path: "/admin/vip3",
              name: "vip3",
            },
          ],
        }
      ]
    }
  }
}
</script>

    # 4. 说明

    al-form Attributes

    参数 说明 类型 可选值 默认值
    menuList 菜单的列表项 Array — 图标

    Slot

    插槽处 说明 name
    al-menu标签内 menu-item内的插槽，用以显示自定义的内容 menuItem
    al-menu标签内 submenu内的插槽，用以显示自定义的内容 submenu
